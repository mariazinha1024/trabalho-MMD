Para o projeto, escolhemos o **Módulo 2**:
```
## Opção 2 – Módulo de custos de manutenção
- Criar dicionário de custos:

  python
  custos = {"troca de óleo": 120.0, "limpeza": 60.0, "troca de rolamento": 300.0}
  
- Cada nova manutenção registrada soma no custo da máquina.
- Gerar relatório `custos_manutencao.txt` com custo por máquina e total geral.
```
E para o restante do código, utilizamos o fornecido no escopo do enunciado do trabalho.
